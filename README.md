# AI 기업분석 자동화 시스템
**한화투자증권 | AI 분석 플랫폼**

DART 전자공시 기반 유동성 탐지 & 우량기관 발굴 플랫폼

---

## 📌 개요

KOSPI·KOSDAQ 상장사의 재무 데이터를 분석하여 **현금성 자산이 풍부한 우량기관을 발굴**하고, 금융상품 영업 우선순위를 자동으로 산출하는 웹 기반 내부 플랫폼입니다.

---

## 🗂 주요 기능

### 1. AI 기업분석
- KOSPI·KOSDAQ 100개 상장사 유동성 현황 일괄 조회
- 유동비율 기준 우선영업 / 일반관찰 / 유동성경보 자동 분류
- 업종별 기업 분포 및 유동비율 구간별 히스토그램 시각화
- 기업명·종목코드·업종 통합 검색
- DART 공시 페이지 바로가기 연동

### 2. 우량기관 발굴
- 현금성 자산 규모 기준 기업 순위 제공
- 업종별 현금성 자산 합계 차트
- 현금 보유 구간 분포 시각화
- 금융 / 제조·산업 / IT·플랫폼 업종 필터

### 3. 영업 캘린더
- 복합 점수 기반 주간 영업 우선순위 자동 산출 (매주 갱신)
- 복합 점수 구성

| 항목 | 가중치 | 설명 |
|---|---|---|
| 현금성 자산 규모 | 40% | 전체 기업 대비 정규화 |
| 유동비율 | 25% | 높을수록 투자 여력 ↑ |
| 발행공시 가산점 | 20% | 당주 공시 기업 우선 |
| 미접촉 기간 | 15% | 최대 8주 기준 |

- 접촉 이력 기록 (담당자 / 유형 / 반응 / 메모)
- 담당자별 접촉기록 조회

### 4. 발행공시 알람
- 채무증권·지분증권 등 자금유동성 관련 발행공시 실시간 모니터링
- DART Open API 연동 지원
- 유형별 필터 (채무증권 / 지분증권 / 소액공모)

## 🚀 사용법

### 기본 사용 (별도 설치 없음)
서비스 URL에 접속하면 바로 사용 가능합니다.
```
https://lhwoo9504-commits.github.io/ver8/
```

### DART API 연동 (선택)
실시간 발행공시 데이터를 받으려면 DART API 키가 필요합니다.

1. [DART Open API](https://opendart.fss.or.kr) 회원가입 후 API 키 발급
2. 화면 좌측 하단 **발행공시 알람** 패널 클릭
3. 하단 입력창에 API 키 입력 후 **연동** 버튼 클릭
4. 이후 자동으로 최근 7일 발행공시 데이터 조회

> API 키 미입력 시 샘플 데이터로 동작합니다.

### 영업 우선순위 활용
1. 사이드바에서 **영업 캘린더** 클릭
2. 이번 주 TOP 10 기업 목록 확인
3. 기업 선택 후 접촉 이력 및 복합 점수 상세 확인
4. 접촉 후 **담당자 / 유형 / 반응 / 메모** 입력하여 기록 저장

### 담당자별 기록 조회
사이드바 하단 담당자 이름 클릭 → 해당 담당자가 작성한 전체 접촉기록 확인
